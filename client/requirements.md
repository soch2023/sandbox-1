## Packages
chess.js | Core chess game logic and FEN/PGN handling
react-chessboard | React component for rendering the board
recharts | Visualization for centipawn loss and opening stats
framer-motion | Smooth UI transitions and animations
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind classes

## Notes
Stockfish engine will be loaded from a public CDN (https://cdnjs.cloudflare.com/ajax/libs/stockfish.js/10.0.0/stockfish.js) to avoid complex worker bundling issues.
Opening Explorer uses the public Lichess Masters API.
